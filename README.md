# Play_Music

O projeto Play Music é um simples tocador de música escrito em Python usando a biblioteca Pygame. Ele foi desenvolvido como parte de uma aula do curso "Python" do Curso em Vídeo ministrado pelo professor Gustavo Guanabara.

## Como usar

1. **Pré-requisitos:**
   - Python instalado
   - Pygame instalado (`python3 -m pip install -U pygame --user`)
  
2. **Clone o repositório em sua máquina local:**
   - `git clone https://github.com/rose-del/Play_Music.git`

3. **Navegue até o diretório do projeto:**
   - `cd Play_Music`

4. **Execute o script Python:**
   - `python3 desafio21.py`

4. **Controles**:
   - Pressione Enter para encerrar a reprodução e fechar o programa.
